## 🔬 Projects

### ⚡ QuantEdge · AI Trading Signal System &nbsp; `Capstone`

> Real-time signal system covering **US and Indian equity markets** — three-model ensemble with SHAP-driven explainability and live sentiment analysis.
>
> **What makes it different:** The system is designed to *withhold* signals when confidence is low. A trading system that knows when not to guess is more useful than one that always outputs something.

**Architecture:**
- 🔷 **Three-model ensemble** — LSTM + XGBoost + Transformer with confidence-gated output
- 🔍 **SHAP explainability** — every signal is interpretable, not just a black-box prediction
- 📡 **Live sentiment pipeline** — integrating financial news and social signal analysis
- 📈 **Backtesting module** — Sharpe ratio, max drawdown, win-rate metrics
- 🖥️ **FastAPI backend** — serving signals with confidence intervals
